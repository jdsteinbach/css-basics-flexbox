flex-direction

Set the axis that flex properties work along

  • row — horizontal
  • column — vertical
  • row-reverse
  • column-reverse

flex-wrap

Allow children to wrap inside the flex container

  • nowrap
  • wrap
  • wrap-reverse

flex-flow

Shorthand for flex-direction & flex-wrap

  • row wrap
  • column wrap-reverse
  • row-reverse nowrap
